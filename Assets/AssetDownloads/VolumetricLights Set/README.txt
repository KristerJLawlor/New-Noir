Setup
-----

This collection of lights make use of Volumetric Lights asset. Make sure you have it imported in your project. You can learn more about Volumetric Lights here: https://assetstore.unity.com/packages/vfx/shaders/volumetric-lights-2-234539

Please follow these steps according to your current rendering pipeline to correctly setup the asset:

a) Built-in pipeline
Make sure Volumetric Lights asset is present in your project and you're ready to use the collection of lights.

b) URP (Universal Rendering Pipeline)
The asset materials and scenes are designed for the built-in pipeline. Use the Rendering Pipeline Converter to convert the Built-in materials to the URP materials. Here's more information about the Rendering Pipeline Converter: https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@12.0/manual/features/rp-converter.html



How to use this asset?
----------------------

The asset includes 7 demo scenes, check them to get an overview of the light models included.
And there're +65 prefabs located in the Prefabs folder. Just drag & drop them into your scene.

Every prefab has a volumetric light child. Select it and customize at will.
Every volumetric light uses a volumetric profile located in the Volumetric Light Profiles folder.
